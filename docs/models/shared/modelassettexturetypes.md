# ModelAssetTextureTypes

When training model assets these are the texture types available to use.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `ALBEDO`       | ALBEDO         |
| `NORMAL`       | NORMAL         |
| `ROUGHNESS`    | ROUGHNESS      |
| `DISPLACEMENT` | DISPLACEMENT   |
| `HDRP_MASK`    | HDRP_MASK      |
